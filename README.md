# KCFinder Documentation

Portal oficial de documentación para el ecosistema mantenido de KCFinder:

- núcleo standalone PHP;
- adaptador oficial para Laravel;
- bundle oficial para Symfony y Flysystem;
- tema opcional basado en Bootstrap 5.

## Desarrollo local

```bash
npm ci
npm run docs:dev
```

La compilación de producción se valida con:

```bash
npm run docs:build
npm run docs:preview
```

El sitio se publica automáticamente mediante GitHub Actions en [GitHub Pages](https://krma-cl.github.io/kcfinder-docs/).
